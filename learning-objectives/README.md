# Learning Objectives

The learning goals and objectives that the CourseKata materials (*Statistics and Data Science: A Modeling Approach*) are designed around, and that its embedded assessment items are aligned to.

If you are evaluating whether CourseKata's learning objectives overlap with your own course's, these documents pair well with the [topic alignment guide](../topic-alignment/README.md), which maps traditional intro-stats topics to CourseKata chapters.

- [learning-objectives.md](learning-objectives.md) — **current version**: the full hierarchy of goals, sub-goals, and objectives, with the book chapters that address each objective (book v5+ chapter numbering, verified against v7.2)
- [learning-objectives.csv](learning-objectives.csv) — the same content in flat, machine-readable form (one row per objective), with numeric objective IDs and the number of assessment items aligned to each objective; useful for filtering, searching, or building your own crosswalk
- [learning-objectives-book-v4.md](learning-objectives-book-v4.md) — archived version 4.0, with chapter references to older versions of the book
- `Learning Goals_ Assessment Objectives 4.0.docx` — original Word document the archived version was converted from
- `learning objective codebook.xlsx` — source spreadsheet for the current version; also contains the objective-to-assessment-item alignment (which specific items assess each objective) and a list of non-assessment (instructional) items
